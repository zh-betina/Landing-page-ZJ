## Responsive (desktop-first) Landing Page

Design: Żaneta Jażdżyk (@nettecode).

###Scrips:

For live preview:
  1) `npm install`
  2) `npm start`

For development:
(Please, change the file name in the <link></link> for .css file from style.css
to general.css)

  1) `npm install`
and then in seperate windows/tabs:
  2)
    a) `npm run sass-watch`
    b) `npm run live-dev`
